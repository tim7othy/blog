需要注意的是，<canvas> 标签只有两个属性—— width和height。这些都是可选的，并且同样利用 DOM properties 来设置。当没有设置宽度和高度的时候，canvas会初始化宽度为300像素和高度为150像素。该元素可以使用CSS来定义大小，但在绘制时图像会伸缩以适应它的框架尺寸：如果CSS的尺寸与初始画布的比例不一致，它会出现扭曲。因此我们最好在HTML文件中嵌入<canvas>元素时就在标签上指定画布的大小。

用路径绘制其他形状

矩形之外的其他形状canvas没有提供默认的API去绘制，我们需要通过自己绘制路径来完成。路径是通过不同颜色和宽度的线段或曲线相连形成的不同形状的点的集合。一个路径，甚至一个子路径，都是闭合的，因此我们通过路径绘制完一个形状之后，可以选择填充路径，来为图形上色。

通过路径绘制一个图形需要通过如下步骤：

  1. 创建路径起始点
  2. 使用画图命令去画出路径
  3. 把路径封闭
  4. 通过描边或填充路径区域来渲染图形
